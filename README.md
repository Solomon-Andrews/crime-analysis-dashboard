# Crime Analysis Dashboard

An interactive Power BI dashboard analyzing crime data to uncover patterns in offense 
types, victim demographics, and case timing.

## 📊 Overview

This dashboard analyzes **549K+ crime records**, categorizing offenses using the NIBRS 
(National Incident-Based Reporting System) classification to surface trends across 
crime type, victim age, and day of occurrence.

## 🔍 Key Insights

- **Crimes Against Property** account for **64.7%** of all recorded offenses — more 
  than double Crimes Against Persons (26.4%) and Crimes Against Society (8.9%)
- **Burglary of Vehicle** is the single leading offense type, followed by Theft and 
  Criminal Mischief
- Case volume is fairly evenly distributed across the week, with **Friday (15.1%)** 
  seeing the highest share of reported cases
- The majority of victim age data was marked "Not Applicable," suggesting many 
  offenses are property-related rather than person-related — consistent with the 
  category breakdown above

## 📈 Visualizations

- Gauge KPI — total offense count
- Bar chart — top offense types by NIBRS code
- Donut chart — case distribution by day of week
- Pie chart — offense breakdown by NIBRS category
- Bar chart — offense count by victim age range

## 🛠️ Tools Used

- **Power BI Desktop** — data modeling, DAX measures, visualization
- Dark theme with a consistent pink-toned color palette across all visuals
